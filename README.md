# igame-ssl-archive

## 简介

igame-ssl-archive 是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态 HTML 文件的集中存储与分发点。

## 目录结构

```
igame-ssl-archive/
├── index.html          # 仓库导航页（可选）
├── pages/              # 存放归档的 HTML 页面
│   └── ...
├── assets/             # 页面依赖的静态资源（CSS、JS、图片等）
│   └── ...
├── README.md           # 本文件
└── LICENSE             # 许可证文件
```

- **pages/**：各独立 HTML 页面存放目录，可按时间或主题组织子目录。
- **assets/**：页面引用的公共资源，确保相对路径引用正确。

## 页面归档说明

1. 所有归档页面均为独立 HTML 文件，可直接通过浏览器打开。
2. 页面资源（样式、脚本、图片）应尽量内联或存放于 `assets/` 目录，避免外部依赖。
3. 归档内容以静态快照形式保存，不保证实时更新或与原始来源一致。
4. 提交新页面时，请确保文件命名清晰，并在 `index.html` 或目录内添加必要链接。

## 维护说明

- 本仓库由维护者定期或按需更新，新增页面将经过基础检查。
- 如需贡献或提交页面，可通过 Pull Request 或 Issue 联系维护者。
- 仓库内容遵循开源许可证（详见 LICENSE 文件），使用前请确认合规性。
- 不鼓励存放涉及隐私、侵权或违反法律法规的内容。
